# Ayuda

Carpeta para recursos de referencia del proyecto: papers, artículos, cheatsheets, notas metodológicas o cualquier documentación de apoyo que no forme parte del código ni de los datos.

**Ejemplos de contenido habitual:**
- Papers o artículos en los que se basa el análisis
- Cheatsheets de librerías (pandas, sklearn, etc.)
- Notas sobre decisiones metodológicas
- Guías internas del equipo

> Esta carpeta no se publica ni se incluye en el paquete. Es solo un espacio de trabajo local.
